Isometric 3D maps based on OpenStreeetMap data

[de] Die folgenden Orte wurden bisher testweise als isometrische 3D-Karte aus Openstreetmap gerendert:
  • Berlin (Ausschnitt) (ca. 25 MB, Letztes Update: 2010-04-17)
  • Braunschweig (Ausschnitt) (ca. 20 MB, Letztes Update: 2010-04-17)
  • Bremen (Ausschnitt) (ca. 27 MB, Letztes Update: 2010-04-17)
  • Hamburg (Ausschnitt) (ca. 26 MB, Letztes Update: 2010-04-16)
  • Hannover (Ausschnitt) (ca. 7 MB, Letztes Update: 2010-04-18)
  • Oyten (ca. 6 MB, Letztes Update: 2010-04-16)
  • Rostock (Ausschnitt) (ca. 25 MB, Letztes Update: 2010-04-15)
  • Uetze (ca. 7 MB, Letztes Update: 2010-04-18)
  • Warnemünde (ca. 27 MB, Letztes Update: 2010-04-15)

The rendering scripts are based on the scripts from http://osm.kyblsoft.cz/3dmapa/.
The source code of my modified osm2pov script is available on Github (licensed under GPLv3). Detailled instructions are coming soon on this website.
